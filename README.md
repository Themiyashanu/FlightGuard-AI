# ✈️ Aviation Risk Prediction Model

## 🎯 Project Overview
A complete machine learning project for predicting aviation accident risk levels using historical Kaggle aviation crash data.

## 🚀 Features
- **Data Analysis**: Structured preprocessing and saved visual insights
- **Machine Learning**: Trains and compares Random Forest, Gradient Boosting, and Logistic Regression
- **Risk Prediction**: Predicts risk probability and class (Low/Medium/High) for new flight scenarios
- **Interactive Dashboard**: Streamlit app for real-time risk checks and charts
- **Professional Visualizations**: Auto-generated plots in `images/`

## 📊 Dataset
- **Source**: Kaggle Aviation Accident Dataset
- **Records**: 5,268 accidents in this local run
- **Time Range**: 1908 onward (from source file)
- **Core Fields Used**: Date, Location, Aircraft Type, Operator, Fatalities, Aboard, Summary

## 🛠️ Technologies Used
- **Python**: pandas, numpy, scikit-learn
- **Visualization**: matplotlib, seaborn, plotly
- **Dashboard**: Streamlit
- **Machine Learning**: Random Forest, Gradient Boosting, Logistic Regression

## 📈 Key Results
- **Best Model**: Gradient Boosting
- **Accuracy**: 0.9023
- **AUC Score**: 0.9497
- **Strong Risk Signals**: Mechanical-failure mentions, weather mentions, pilot-error mentions, aircraft category, operator type

## 🚀 Quick Start

### Installation
```bash
pip install -r requirements.txt
```

### Train Model
```bash
python src/train_model.py
```

### Run Dashboard
```bash
python -m streamlit run dashboard/app.py
```

## 📊 Sample Predictions
- **Low Risk Scenario**: Commercial operator, no weather/mechanical/pilot flags -> lower predicted probability
- **High Risk Scenario**: Military operation with adverse weather + mechanical/pilot flags -> higher predicted probability

You can also run CLI predictions:
```bash
python src/predict.py --year 2020 --month 6 --aboard 140 --is_commercial 1 --weather_mentioned 0
```
